1. Overview

This API provides management tools of an airport monitoring information system

1.1. Version information

Version : 1.0

1.2. URI scheme

Host : localhost:8081
BasePath : /

1.3. Tags

  • Aircraft Entity : Simple Jpa Repository

  • AircraftType Entity : Simple Jpa Repository

  • Apron Entity : Simple Jpa Repository

  • Area Entity : Simple Jpa Repository

  • Arrival Entity : Simple Jpa Repository

  • Atv Entity : Simple Jpa Repository

  • Departure Entity : Simple Jpa Repository

  • Equipment Entity : Simple Jpa Repository

  • Event Entity : Simple Jpa Repository

  • Flight Entity : Simple Jpa Repository

  • GeoElement Entity : Simple Jpa Repository

  • Model Entity : Simple Jpa Repository

  • ModelElement Entity : Simple Jpa Repository

  • Stand Entity : Simple Jpa Repository

  • Terminal Entity : Simple Jpa Repository

  • TraitmentEquipement Entity : Simple Jpa Repository

  • admin-controller : Admin Controller

  • aircraft-type-history-controler : Aircraft Type History Controler

  • arrival-history-controler : Arrival History Controler

  • departure-history-controler : Departure History Controler

  • geo-element-history-controler : Geo Element History Controler

  • profile-controller : Profile Controller

2. Authentication

2.1. Header

All services are rights protected, an authentication token must always be provided in the request header as follows

Authorization: token

2.2. Token

Fetching the requested authentication token is done through the below service
